## 海雾客观预报方法

* 指标1: 温度露点差t2m_td
* 指标2: 露点海温差td_sst
* 指标3: 露点海温差>0并且温度露点差＜0.8摄氏度为海雾区，露点海雾差越大，海雾越明显
* 指标4:  露点海温差>0但温度露点差≥0.8摄氏度为非海雾区
